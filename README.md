Guess Game

Features

Randomized Target Number: Each game generates a new number for a unique experience.
Feedback for Each Guess: Shows whether your guess is too high, too low, or correct.
Configurable Difficulty (if added): Options for different ranges or guess limits.

Goal

Guess the randomly selected number in the fewest tries possible!
